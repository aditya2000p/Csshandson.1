what is css?

  1. css stand for cascading style sheet.
  2. css describe how HTML are to be displayed on screen , paper, or in other media.
  3. css is not nessesary css save is lot of work. it can control the layout of multiple web pages all at once
  4. External stylesheets are store in css files.

Css can be added to HTML documents in 3 ways:

  • inline - by using style attributes inside HTML elementsl
  • internal - by using a style element in the head section
  • External - by using a link element to link to an external css file.
  • The most common way to add css, is to keep the style in externl css file. However . in thise tutorial we will use inline and internal styles, because thise is easier to demonstrate . and easier for you to try it your self

when comparing CSS class vs ID,the difference is that CSS class applies a style to multiple element . ID ,on the other hand ,applies a style to one unique element. ID is also special in that you can use a special URL to link directly to an element and it's used by javascript.

In css ,selector are used to target a specific element or range of element on web page. once an element has been targeted, a style or set of styles can be applied to the element.

There is a wide range of selector avilabile. Two of the most commonly used are class and ID . both are used to target elements to which a style should be applied.
